# Ver.2.1.4 問題形式・共通基盤監査結果

## Ver.2.1.4の追加監査

- 問題データが `app.js` から `data/k9-course.js` へ分離されていること
- 9級コースが共通レジストリへ登録され、保存キーが `KANJI9_SAVE_V2` のままであること
- 全20日で5形式が各2問、通常学習が合計10問になること
- 全問題バンク320問のID、正解、3択、対象漢字、形式を検証すること
- 既存の読み100問・書き100問のIDが保持されること
- 新形式の使い分け40問・対になる漢字40問・漢字の仲間40問を解決できること
- 旧保存データと復習履歴を保持し、形式別成績を新規追加できること
- 連打防止、戻る、バックアップ、iPhone表示、A4印刷と全既存機能が回帰すること

以下はVer.2.1.3までの監査内容です。

## Ver.2.1.3の追加監査

- 同じ選択肢を連打しても回答結果が1件だけ記録されること
- 「つぎへ」を連打しても1問だけ進むこと
- 学習完了処理を重ねて呼んでもEXP・星・ガチャ券が一度だけ付与されること
- ガチャボタンを連打しても券の消費と履歴追加が一度だけであること
- 学習途中の画面内戻る・ブラウザー戻るで離脱確認が働くこと
- Ver.2.1.2の保存データ、監査済み問題履歴、暗証番号、装備を保持すること
- JSON書き出しデータを復元でき、復元前データが直前バックアップへ残ること
- iPhone幅で横スクロールや入力時の意図しない拡大がないこと
- A4印刷が1ページ4字、8字で2ページになること
- `version.json` をキャッシュ無効で確認し、新版公開時に更新案内を出せること

以下はVer.2.1.2で実施し、Ver.2.1.3でも回帰確認する問題・印刷監査です。

## 監査対象

- 実装済みカリキュラム：20日
- 1日あたり：読み5問＋書き5問
- 合計：200問
- 漢字図鑑：91字

## 問題監査で修正した内容

1. すべての問題で正解番号が `0`（一番上）に固定されていたため、学習開始ごとに正解位置を上・中央・下へ分散させました。
2. 漢字を1字だけ表示する読み問題は複数の読み方が成立する場合があるため、すべて短い例文の中で問う形式へ変更しました。
3. 書き問題も例文の一部を `□` にし、空欄へ入る漢字を選ぶ形式へ変更しました。
4. 各問題の選択肢が3つあり、重複がなく、正解が1つであることを自動監査しました。
5. 20日分の問題IDがすべて一意であることを確認しました。

## 漢検9級範囲との照合

漢検9級は小学校2年生修了程度で、対象は小学校1・2年生の累計240字です。
旧版の91字には範囲外の14字が含まれていたため、次のように修正しました。

| 旧版の範囲外漢字 | Ver.2.1.2の範囲内漢字 |
|---|---|
| 飲 | 肉 |
| 短 | 形 |
| 暗 | 黒 |
| 葉・根・実 | 木・林・森 |
| 駅 | 市 |
| 仕・事・働 | 工・買・会 |
| 座・開・閉・待 | 行・帰・来・休 |

出典：

- 日本漢字能力検定協会「各級の概要」  
  https://www.kanken.or.jp/kanken/grades/overview/
- 文部科学省「学年別漢字配当表」  
  https://www.mext.go.jp/a_menu/shotou/new-cs/youryou/syo/koku/__icsFiles/afieldfile/2016/10/27/1234920.pdf

## A4印刷監査

- 用紙：A4縦
- 印刷余白：8mm
- 1ページ：4字
- 各ページにタイトル、Day、日付、名前欄、ページ番号を表示
- 各漢字に「なぞる」「見て書く」「見ないで」の3段を配置
- 8字選択時にA4ちょうど2ページになることをChromium PDFで確認
- PDFページサイズがA4（594.96×841.92pt）であることを確認
- 文字切れ、枠の分断、余分な空白ページがないことをページ画像で確認

## 保存データ互換性

保存キー `KANJI9_SAVE_V2` は変更していません。
Ver.2.1.1形式のデータを読み込み、次が保持されることを確認しました。

- 現在Dayと完了日
- 星と学習統計
- ガチャ券と所持アイテム
- 保護者用暗証番号
- 内容が変わっていない問題の復習履歴

差し替え対象の問題だけは、旧漢字の成績を新漢字へ誤って引き継がないよう、初回起動時に復習履歴から外します。

## 現在の収録範囲に関する注意

Ver.2.1.2の収録漢字は91字で、漢検9級の対象240字すべてを網羅しているわけではありません。
現在の実装は20日・200問です。240字すべてを学習できる完全版への拡張は、次版以降の別工程です。
