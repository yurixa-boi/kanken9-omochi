# CHANGELOG

## Ver.2.1.5

- 保護者ページでリアルごほうび名・アイコン・ドロップ率を設定可能
- ガチャ1回につき、リアルごほうび交換券か通常シールのどちらか一方を抽選
- ごほうびの確率合計を100％以下に制限し、残りを通常シールの確率として表示
- 交換券を子ども側のごほうび画面に保存し、保護者が使用済みに変更可能
- 獲得日時・使用日時・獲得元を保存
- ごほうび設定を削除しても、獲得済み交換券の名称・アイコン・履歴を維持
- 従来の星との交換、`KANJI9_SAVE_V2`、ガチャ、シール、全学習機能を維持

## Ver.2.1.4

- 9級問題データを `app.js` から `data/k9-course.js` へ分離
- 8級・7級を同じ形式で登録できるコースレジストリを追加
- 問題オブジェクトに `courseId` と `format` を追加
- 読み・書き取り・使い分け・対になる漢字・漢字の仲間の5形式に対応
- 1回10問を各形式2問ずつで構成し、形式の順番も毎回分散
- 読み・書きの既存ID `d?r?`・`d?w?`を維持
- 新形式ID `d?u?`・`d?p?`・`d?b?`を追加
- 全問題バンクから問題IDを解決するよう復習処理を拡張
- 形式ごとの正解・ミスを保存し、保護者ページへ正答率を表示
- 将来の級切り替え用にコース別保存キーをデータ定義へ追加
- 既存の `KANJI9_SAVE_V2` と全機能を維持

## Ver.2.1.3

- 回答直後に選択肢をロックし、回答記録・星・結果が二重加算されない処理を追加
- 「つぎへ」、ガチャ抽選、暗証番号4桁処理の多重実行を防止
- 学習完了処理に一度限りのガードを追加し、EXPとガチャ券の二重付与を防止
- 画面履歴を導入し、ブラウザー戻る・iPhoneスワイプバックをホーム復帰へ統一
- 学習途中の離脱には確認を表示し、キャンセル時は問題画面を維持
- iPhoneのセーフエリア、狭幅、横向き、入力ズーム、モーダル高を調整
- A4縦・8mm余白・4字/ページを維持し、フォントと描画確定後に印刷
- 学習データのJSON書き出し、ファイル復元、直前バックアップ復元を追加
- 読込JSONの検証、5MB制限、危険なプロパティの除外を追加
- `version.json` の更新確認と版番号付きCSS・JavaScript・画像URLを追加
- Ver.2.1.2の監査済み問題履歴が再移行で削除される条件を修正
- `KANJI9_SAVE_V2` と全既存機能を維持
