# おもち先生と学ぶ 漢検9級チャレンジ

Cloudflare Pages対応の、小学2年生向け漢検9級学習ゲームです。

## Ver.2.1.5

- 保護者が実際のごほうびとガチャ用ドロップ率を設定
- ガチャでは交換券または通常シールのどちらか一方を獲得
- 交換券を貯め、保護者ページで使用済みに変更
- 設定削除後も獲得済み交換券の履歴を維持
- 従来の星との交換と `KANJI9_SAVE_V2` を維持

## Ver.2.1.4

- 9級のカリキュラム・読み・例文・新形式データを `data/k9-course.js` へ分離
- 級別データを登録する `data/course-registry.js` を追加
- 読み・書き取り・使い分け・対になる漢字・漢字の仲間の5形式に対応
- 通常学習は5形式を2問ずつ、合計10問で出題
- 既存の読み・書き問題IDを維持し、新形式だけ新しいIDを追加
- 保護者ページに形式別の正答率を追加
- 8級・7級は同じデータ構造のファイルを登録して追加できる共通基盤へ変更
- `KANJI9_SAVE_V2`、復習履歴、図鑑、ガチャなど既存機能を維持

## 問題データの構成

```text
data/
├── course-registry.js   # 全級共通の登録処理
└── k9-course.js         # 9級の問題・読み・例文・5形式データ
```

8級・7級を追加するときは、同じ形の `k8-course.js`、`k7-course.js` を登録します。画面・採点・復習処理は `app.js` の共通エンジンを利用します。

## 保存データについて

保存キーは従来どおり `KANJI9_SAVE_V2` です。
Ver.2.1.2までの学習進捗、星、ガチャ、シール、保護者暗証番号を残したまま更新します。
範囲外漢字から差し替えた問題の整理はVer.2.1.2より前の保存データにだけ実行し、Ver.2.1.2以降の履歴は保持します。

保護者ページからJSONファイルへ書き出せます。復元前・初期化前の現在データは `KANJI9_BACKUP` へ自動退避され、「直前のデータへ戻す」で入れ替えられます。

念のため、更新前の公開ファイル一式を別フォルダへ保存してから差し替えてください。

## 公開方法

このフォルダ内のファイルと `assets` フォルダを、GitHubリポジトリのルートへアップロードしてください。
Cloudflare PagesのFramework presetは `None`、Build commandは空欄、Build output directoryは `/` のままで動作します。`version.json` も必ず一緒に公開してください。

## 主な確認項目

1. 以前の学習データを残した状態でホーム画面が開くこと
2. 学習後、漢字図鑑の正解数・ミス数・正答率が更新されること
3. 間違えた漢字がミスノート絞り込みと保護者ページへ表示されること
4. 手書きプリントがA4で印刷できること
5. ガチャ、着せ替え、冒険マップ、暗証番号が引き続き動くこと

詳しい監査結果は `AUDIT_REPORT.md` を参照してください。
