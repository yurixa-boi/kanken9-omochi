# おもち先生と学ぶ 漢検9級チャレンジ

Cloudflare Pages対応の、小学2年生向け漢検9級学習ゲームです。

## Ver.2.1.2

- 学習開始ごとに3択の正解位置を上・中央・下へ分散
- 読み・書き問題を例文の中で答える形式へ変更
- 20日分・200問の正答、選択肢重複、問題IDを自動監査
- 収録91字を小学校1・2年生配当漢字の範囲内へ修正
- 実装済み日数に合わせ、ホームの進捗表示を20日へ統一
- A4プリントを1ページ4字で固定し、各ページへ見出しと名前欄を表示
- 8字がA4・2ページになることをChromiumとPDFで検証
- 既存のLocalStorageキー、進捗、星、ガチャ、暗証番号を維持

## 保存データについて

保存キーは従来どおり `KANJI9_SAVE_V2` です。
Ver.2.1.1までの学習進捗、星、ガチャ、シール、保護者暗証番号を残したまま更新します。
範囲外漢字から差し替えた問題だけは、別の漢字へ誤った履歴を移さないため、旧復習履歴から自動的に外します。

念のため、更新前の公開ファイル一式を別フォルダへ保存してから差し替えてください。

## 公開方法

このフォルダ内のファイルと `assets` フォルダを、GitHubリポジトリのルートへアップロードしてください。
Cloudflare PagesのFramework presetは `None`、Build commandは空欄、Build output directoryは `/` のままで動作します。

## 主な確認項目

1. 以前の学習データを残した状態でホーム画面が開くこと
2. 学習後、漢字図鑑の正解数・ミス数・正答率が更新されること
3. 間違えた漢字がミスノート絞り込みと保護者ページへ表示されること
4. 手書きプリントがA4で印刷できること
5. ガチャ、着せ替え、冒険マップ、暗証番号が引き続き動くこと

詳しい監査結果は `AUDIT_REPORT.md` を参照してください。
