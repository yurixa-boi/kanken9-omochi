# おもち先生と学ぶ 漢検9級チャレンジ

Cloudflare Pages対応の、小学2年生向け漢検9級学習ゲームです。

## Ver.2.1.1 Part3-2③

- 単一HTMLから `index.html` / `style.css` / `app.js` へ分割
- 埋め込み画像を `assets/images` に分離・重複排除
- 既存のLocalStorageキーと学習データを維持
- GitHub + Cloudflare Pagesの自動デプロイに対応
- 漢字カード図鑑を正式版へ更新
- 学習結果・正答率・初回学習日を漢字ごとに記録
- ミスノートの苦手漢字を図鑑と保護者ページへ自動反映
- 今日の漢字＋苦手漢字からA4手書きプリントを自動作成
- 図鑑から練習したい漢字を選択可能
- レベル・EXP表示、ガチャアイテムの着せ替えを実装

## 保存データについて

保存キーは従来どおり `KANJI9_SAVE_V2` です。
Ver.2.1-①までの学習進捗、星、ガチャ、シール、保護者暗証番号を残したまま、新しい図鑑用データを自動追加します。

念のため、更新前の公開ファイル一式を別フォルダへ保存してから差し替えてください。

## 公開方法

このフォルダ内のファイルと `assets` フォルダを、GitHubリポジトリのルートへアップロードしてください。
Cloudflare PagesのFramework presetは `None`、Build commandは空欄、Build output directoryは `/` のままで動作します。

## 主な確認項目

1. 以前の学習データを残した状態でホーム画面が開くこと
2. 学習後、漢字図鑑の正解数・ミス数・正答率が更新されること
3. 間違えた漢字がミスノート絞り込みと保護者ページへ表示されること
4. 手書きプリントがA4で印刷できること
5. ガチャ、着せ替え、冒険マップ、暗証番号が引き続き動くこと
